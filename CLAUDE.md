## 项目概述
北京林业大学 CS2（Counter-Strike 2）比赛报名小程序，使用 Python 技术栈开发。作为简历项目，注重代码质量和项目完整性。

## 技术栈
- 后端：Python（框架待定：Flask / FastAPI / Django）
- 前端：待定（Jinja2 模板 / 前后端分离 Vue/React）
- 数据库：待定（SQLite / MySQL / PostgreSQL）
- 部署：待定

## 核心功能（规划中）
